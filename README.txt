PROFESYONEL KAFE MENÜSÜ

DOSYALAR
- index.html : Menü içeriği, kafe adı, adres ve fiyatlar
- styles.css : Tasarım, renkler ve mobil görünüm
- script.js  : Arama ve kategori filtreleme

DÜZENLEME
1. index.html dosyasını VS Code / Not Defteri ile açın.
2. "KAFE ADI" yazan alanları kendi işletme adınızla değiştirin.
3. "000 ₺" ifadelerini gerçek fiyatlarla değiştirin.
4. Adres, telefon, Instagram ve çalışma saatlerini güncelleyin.
5. index.html dosyasına çift tıklayarak tarayıcıda kontrol edin.

ÜCRETSİZ YAYINLAMA
- GitHub Pages: dosyaları repository'ye yükleyip Settings > Pages bölümünden yayınlayın.
- Cloudflare Pages: klasörü GitHub'a yükleyin, Cloudflare Pages ile bağlayın.
- Netlify Drop: klasörü Netlify Drop sayfasına sürükleyip ücretsiz URL alın.

NOT
Google Fonts kullanıldığı için fontların ilk açılışta yüklenmesi internet bağlantısı gerektirir.
Font yüklenmezse sistem fontlarıyla çalışmaya devam eder.
